Другие настройки

(Подробнее) Для задания значения можно использовать постфиксы:
  • `k` КилоБайты
  • `m` МегаБайты = `kk`
  • `g` ГигаБайты = `kkk`
  • `0` для всех архивов

(Подробнее) Для задания значения можно использовать постфиксы:
  • `-1` для бесконечной вложенности (По умолчанию)
  • `0` для отбрасывания всех архивов
  • `1` для входа на один уровень архива
